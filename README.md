# BRTRANSPARENT
> Requisitos para rodar o projeto:

- Instalar o [NodeJs](https://nodejs.org/en/download/)

1: Instalar dependências
```JavaScript
    npm install
```
2: Rodar o projeto
```JavaScript
    npm run dev
```

3: Abrir projeto na porta [3000](http://localhost:3000)
